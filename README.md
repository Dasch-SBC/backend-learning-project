# 💻 Getting Started

Clone the repository:

```bash
git clone https://github.com/Dasch-SBC/backend-learning-project.git
cd backend-learning-project
```

Create a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies:

```bash
pip install flask
```

Run a project:

```bash
python app.py
```
